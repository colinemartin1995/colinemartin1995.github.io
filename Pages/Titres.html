Afin de définir une structuration hiérarchique des paragraphes dans une page on utilise plusieurs niveaux de titre :

Ceci est un titre de type 1

Niveau 1
Pour cela j'utilise la balise <h1>

Ceci est un titre de type 2

Niveau 2
Pour cela j'utilise la balise <h2>

Ceci est un titre de type 1

Niveau 3
Pour cela j'utilise la balise <h3>

Note : Il existe jusqu'a 6 niveau pour les titres, cependant il n'est généralement utilisé que le h1, h2 et h3
